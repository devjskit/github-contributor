# github-contributor

- The listings show all the contributors.
- Sort by contributions number.
- Sort by commit date.

## Installment

```bash
$ npm i github-contributor --save-dev
```

```json
"devDependencies": {
  "github-contributor": "*",
},
"scripts": {
  "contributor": "github-contributor",
}
```
